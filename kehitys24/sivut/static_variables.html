Staattiset muuttujat

Staattiset muuttujat ovat keino välittää muuttujien kautta tietoa koko sovelluksesi sisällä.

  • Lisää uusi GameObject -> nimeä Globals
  • Lisää tee uusi skripti -> Globals.cs
  • isää skriptin sisälle vain tarvitsemasi globaalit staattiset muuttujat 
    public class Globals : MonoBehaviour
{
    public static int pisteet;
}
  • Voit käyttää nyt pisteitä suoraan toisessa luokassa näin: 
    Globals.pisteet = 0;
